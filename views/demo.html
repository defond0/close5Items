Items API

This lightweight API has one simple route to cover your basic needs:
'api/v1/items' get an item by id, get all items, get user items, get items within 50 miles of your location

Docs

All requests are GET requests to the route api/v1/items:
To specify what sort of items you want append a query string with the following values;


Required/Unique key(s)

sortBy:
Can be set set to 'price' or 'date'. Will return a list of all items sorted by either price or date created. Default is date
loc:
Returns all items within range of specified co-ordinates. Must be set in the following format [longitude,latitude]
userId:
Returns all items associated with the specified user ID
itemId:
Returns Item with specified ID

Optional key(s)

ascending:
Sets sort order for all sorted lists. If set to false will sort results by descending values. Default is true

Examples

To get items send a GET request to http://localhost:8080/api/v1/items?




Example Ajax request:
//Url with Query String = localhost:8080/api/v1/items?
$.ajax({
  url:'/api/v1/items',
  method:'get',
  data:Form Input;,
});
